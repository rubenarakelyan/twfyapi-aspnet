# twfyapi-aspnet

This repo provides ASP.NET wrappers for the TheyWorkForYou.com and OpenAustralia.org.au APIs.

### Contribution Guidelines

When contributing please consider the following guidelines:

- Please conform to the code style of the project.
- All methods and classes must contain docblocks.
- When planning a pull request to add new functionality, it may be wise to [submit a proposal](https://github.com/rubenarakelyan/twfyapi-aspnet/issues/new) to ensure compatibility with the project's goals.
